Demo Guide

This page explains the demo process and provides quick links to the two remote nodes and downloadable sample datasets.

Remote endpoints

  • Remote 1 (served under /remote1)
  • Remote 2 (served under /remote2)

Download demo data

These CSV files are hosted under /static/demo_data on the central server.

  • multi_missing_central.csv
  • multi_missing_remote1.csv
  • multi_missing_remote2.csv
  • single_missing_binary_central.csv
  • single_missing_binary_remote1.csv
  • single_missing_binary_remote2.csv
  • single_missing_continuous_central.csv
  • single_missing_continuous_remote1.csv
  • single_missing_continuous_remote2.csv

What each dataset shows

  • Multi-missing imputation: multiple variables have missing values; assume column 1 has missing binary values, and column 7 has missing continuous values.
  • Single-missing (binary): only column 1 is missing and is binary.
  • Single-missing (continuous): only column 1 is missing and is continuous.

Use these datasets with the Start Job page to run a SIMI/SIMICE demo across the central and two remotes.
